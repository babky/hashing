The description of the problem

Create a program which will sort a file stored on a computer's hard drive. Assume that the input file is stored in the current directory and named data.txt The file data.txt is a text file. Each row consists of the 64-bit unsigned values - a key and a value. In the sorted output file data.out the keys are ascending and for each key the minimum value associated with the key is outputted.

Generating the test data

$/afs/ms.mff.cuni.cz/u/k/koucky/ds1/2015/hw1/gen-data -s XX >/tmp/data.txt

generates the whole input data.

$/afs/ms.mff.cuni.cz/u/k/koucky/ds1/hw1/gen-data -s XX --test >/tmp/data.txt

generates the data for creating your program - only 128 records.

For generating shorter inputs for testing and debugging you can also use

$/afs/ms.mff.cuni.cz/u/k/koucky/ds1/hw1/gen-data -s XX --short >/tmp/data.txt
$/afs/ms.mff.cuni.cz/u/k/koucky/ds1/hw1/gen-data -s XX --super-short >/tmp/data.txt

The XX are the two last digits of your student number.
